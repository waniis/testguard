=== Kwanko - Tracking Tags for Advertisers ===
Tags: kwanko
Requires at least: 4.0.26
Tested up to: 5.2.2

WooCommerce plugin that handles the setup of the kwanko tracking tags for the advertisers.

== Description ==

WARNING: This is a WooCommerce plugin. It does nothing if WooCommerce is not installed.

Tested WooCommerce versions: 2.1.12 > 3.6.4

This plugin makes it easy to install the kwanko tracking tags for the advertisers.

== Installation ==

1. Click on "Add New" button on the plugin page.
2. Upload the plugin zip archive.
3. Activate the plugin.
4. Go to the plugin settings page and configure at least the unijs file and the mclic.
